# PublicAppendix
Public Appendix for the 6th semester project. Consisting of test data. In these folder the following can be found

 - **Antenna_measurements**: The measured antenna s-parameters \& and radiation patterns.
 - **Inventor**: The CAD files used for this project.
 - **Meshing_Firmware**: A separate git repository including the different branches of the developed mesh firmware and network stack.
 - **Monte_carl_sim**: A separate git repository showing all branches of the software used for simulating antenna parameters.
 - **Test**: The raw testdata from the physical tests along with a javascript plotter to genereate the graphs illustrated in chapter 6.


All content including subdirectories can be downloaded by
```console
git clone --recurse-submodules git@github.com:ESD625-Fedtmules-muler/PublicAppendix.git
```
